A: 10 B: 20 C: 30

第一个exchange在多个账户间运行
exchange A B: {
  i = B - A = 10
  B = B - i = 10
  A = A + i = 20
}

exchange A C: {
  i = C - A = 20
  exchange A B( A = 20 B = 10)
  C = C - i = 10
  A = A + i = 40
}

总和不变的原因是，每个账户的操作是串行化的，因此对每一个账户的操作最终都会反映到该账户上
(非串行的操作则可能不会，如对账户A进行一次+5 一次+10,可能只反映出其中一种结果)
